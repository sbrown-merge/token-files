# Design System: Cynergy Wellness

## Typography

- **Headings**: Montserrat (700 Bold)
  - Used for H1, H2, and important UI labels.
- **Body Text**: Open Sans
  - Regular (400) for primary content.
  - Light (300) for secondary details.
  - Base size: 16px
  - Line-height: 1.5
- **Taglines**: Montserrat (Light)

## Color Palette

- **Brand Navy** `#00334E`
  - Use for headers, footers, and primary buttons.
- **Sky Blue** `#8BBBE5`
  - Use for logo accents and highlights.
- **Steel Blue** `#7092BE`
  - Use for secondary elements, taglines, and card borders.
- **Slate Gray** `#4A4A4A`
  - Use for primary body text.
- **Mist Gray** `#F4F7F6`
  - Use for page backgrounds.

## UI Components

### Buttons
  - **Primary**: Solid Brand Navy `#00334E` with white text and 4px border radius.
  - **Secondary**: Ghost style outlined button with Steel Blue `#7092BE` border and text. No fill.
### Cards
  - Use modular cards with a thin 1px Steel Blue `#7092BE` border.
  - Background: White `#FFFFFF`.
### Hero Sections
  - Always use a gradient overlay transitioning from Navy to Sky Blue over images.
### Spacing
  - Prioritize large margins and ample whitespace to emphasize trust and clarity.

## Framework Context

- This project is built using the **MUI** component library (`mui.com`).
- Ensure all generated UI maps to standard MUI layouts: Containers, Boxes, and Grids.
